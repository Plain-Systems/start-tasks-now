No motivation to start tasks

If motivation is completely absent and waiting doesn’t help, the problem isn’t discipline. This page shows how to start anyway using a mechanical trigger.

What this is

  • A way to start without motivation
  • No discipline or willpower
  • No habits to build
  • Designed for low-energy days

What to do right now (2 minutes)

  1. Choose one visible action you can do without thinking.
  2. Do it once.
  3. Stop immediately. Motivation is not required.

Why motivation doesn’t work

Motivation is unreliable. Waiting for it creates delay and frustration.

Starting becomes possible again when you remove the need to feel ready.
